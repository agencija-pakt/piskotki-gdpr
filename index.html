PISKOTKI GDPR - SKRIPTA

Za spletne strani in aplikacije

Končno! Celovita rešitev piškotkov za spletne strani (pripravljeno po smernicah GDPR uredbe).

25. maja 2018 je začela veljati splošna uredba GDPR. V podjetju Agencija Pakt d.o.o. smo za lastne potrebe izdelali napredno rešitev za piškotke za spletne strani in aplikacije, katero celotni Sloveniji ponujamo odprtokodno in brezplačno!

Produkt je sestavljen iz skripte (plugin) za popolno funkcionalnost piškotkov za spletne strani. Prav tako pa ponujamo predloge splošnih pogojev, kjer se besedila avtomatsko prilagodijo vašemu naslovu spletne strani. Gre za celovito, varno in najpreprostejšo rešitev na trgu.

!! Pogoj brezplačne uporabe piskotkiGDPR je navedba povezave avtorja (piskotkiGDPR). Povezave so že dodane, tako da vam ni potrebno skrbeti, le izbrisati jih ne smete.

Piškotki

Piškotke delimo na tri glavne kategorije, glede na kategorijo lahko funkcionalnosti povezane s piškotki izklopimo. Npr. če se uporabnik ne strinja s piškotki družbenih omrežij, lahko glede na ime piškotka opt-in-social izklopimo dostop do YouTube posnetka.


cookie-notice-accepted

Piškotki potrebni za osnovno delovanje strani.


opt-in-social

Piškotki družbenih omrežij.


opt-in-remarketing

Piškotki za ponovno trženje.

Pogoji

Zraven skripte za vklop in izklop piškotkov na vaši strani, vam ponujamo še pametne pogoje uporabe. Pogoji se avtomatsko prilagodijo vaši spletni strani oz. imenu vaše strani.

Vse kar morate storiti je, da namestite piskotkigdpr.min.js (glejte odsek namestitev) kodo v glavo vaše strani in nato prilepite kodo pogojev (splosni pogoji, zasebnost, piskotki) na vašo spletno stran. Ime vaše spletne strani se bo samodejno preneslo v celotno besedilo pogojev.

Pogoje dodajte v section ali div s class imenom piskotki:

      <div class="piskotki">
        ...
      </div>

V pogojih: piškotki, boste našli odsek Nastavitve piškotkov, kjer lahko uporabniki naknadno spreminjajo nastavitve piškotkov.

Namestitev

1. Prenos

Skripto lahko prenesete → tukaj (zip) ←

2. HTML/Javascript

Uporabite piskotkigdpr.min.js za produkcijo in piskotkigdpr.js za razvoj.

Kopirajte kodo script in style v glavo vaše strani.

<html>
<head>
	  <script src="piskotkigdpr-min.js"></script>
	  <link rel="stylesheet" type="text/css" href="piskotkigdpr-min.css">
</head>
...

Ali preko CDN (preskočite korak 1.) 
<html>
<head>
	  <script src="https://cdn.jsdelivr.net/gh/agencija-pakt/piskotki-gdpr/piskotkigdpr.min.js"></script>
	  <link rel="stylesheet" type="text/css" href="https://cdn.jsdelivr.net/gh/agencija-pakt/piskotki-gdpr/piskotkigdpr.min.css">
</head>
...

3. Script

Skripto poženete z naslednjim ukazom:

<script>
	/* Nastavitve vtičnika piskotkiGDPR */
	piskotkiGDPR({
		zavesa                : false,
		barvaZavese           : 'rgba(0,0,0,.3)',

		lokacija              : 'spodaj-desno',
		sencaOkna             : '0 10px 20px 0 rgba(0,0,0,.5)',
		zaobljenostOkna       : '20px',
		barvaOzadja           : 'rgba(0,0,0,.7)',
		animacija             : true,

		imeStrani             : 'Ta spletna stran',
		trajanjePiskotka      : 30,

		barvaIkone            : '#fff',
		barvaPisave           : '#fff',
		velikostPisave        : '14px',

		naslov                : 'Spletno mesto uporablja piškotke',

		besedilo              : 'S piškotki zagotavljamo boljšo uporabniško izkušnjo, enostavnejši pregled vsebin, analizo uporabe, oglasne sisteme in funkcionalnosti. S klikom na »Strinjam se« dovoljuješ vse namene obdelave. Posamezne namene  lahko izbiraš na strani -',

		barvaPovezave         : '#fff',
		imePovezavePogoji     : 'Nastavitve',
		povezavaPogoji        : '/piskotki/',
		velikostPovezave      : '14px',

		besediloGumba         : 'Strinjam se',
		barvaGumba            : '#1be195',
		zaobljenostGumba      : '100px',
		barvaGumbaPovezave    : '#4c4c4c',
		velikostGumbaPovezave : '16px'
	});
</script>

Omejevanje

Ključni razlog uredbe piškotkov je, da obiskovalec lahko omeji dostop do raznih funkcionalnosti in tako omejuje dostop do svojih podatkov. Večina skript za piškotke pokaže le obvestilo, vendar so popolnoma brez vsake funkcionalnosti (GDPR takšne prakse več ne dopušča).

piskotkiGDPR imajo tudi možnost omejevanja!

JavaScript

if (preglej('opt-in-social') === true) {
	// Prikaži vsebino
	...
} else {
	// Skrij vsebino
	...
}

PHP

if (isset($_COOKIE['opt-in-social'])) {
	// Prikaži vsebino
	...
} else {
	// Skrij vsebino
	...
}

Omejevanje deluje tudi preko Google Tag Managera.
